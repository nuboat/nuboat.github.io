Developing Web Services on AWS Lambda with Java

Coding > May 2026  ·  Peerapat A  · 

1. To create an execution role

  • Open the roles page in the IAM console.
  • Choose Create role.
  • Create a role with the following properties.
  • Trusted entity – Lambda.
  • Permissions – AWSLambdaBasicExecutionRole.
  • Role name – lambda-role.

The AWSLambdaBasicExecutionRole policy has the permissions that the function needs to write logs to CloudWatch Logs. You can add permissions to the role later, or swap it out for a different role that's specific to a single function.

2. To create a Java function

  • Open the Lambda console.
  • Choose Create function.
  • Configure the following settings:
  • Name – my-function.
  • Runtime – Java 11.
  • Role – Choose an existing role.
  • Existing role – lambda-role.
  • Choose Create function.
  • To configure a test event, choose Test.
  • For Event name, enter test.
  • Choose Save changes.
  • To invoke the function, choose Test.

  • 选择 Save changes。
  • 要 invoke 你的 function，请选择 Test。

Java を使用した AWS Lambda 上の Web Services 開発

May 2026  ·  Peerapat A  ·  ← Coding

1. execution role の作成

  • IAM console で roles ページを開きます。
  • Create role を選択します。
  • 次の properties で role を作成します。
  • Trusted entity – Lambda。
  • Permissions – AWSLambdaBasicExecutionRole。
  • Role name – lambda-role。

AWSLambdaBasicExecutionRole policy には、function が CloudWatch Logs に logs を書き込むために必要な permissions が含まれています。後で role に permissions を追加したり、単一の function 固有の別の role に入れ替えたりすることができます。

2. Java function の作成

  • Lambda console を開きます。
  • Create function を選択します。
  • 次の設定を configure します：
  • Name – my-function。
  • Runtime – Java 11。
  • Role – Choose an existing role。
  • Existing role – lambda-role。
  • Create function を選択します。
  • test event を configure するには、Test を選択します。
  • Event name に test と入力します。
  • Save changes を選択します。
  • function を invoke するには、Test を選択します。

Entwicklung von Web Services auf AWS Lambda mit Java

May 2026  ·  Peerapat A  ·  ← Coding

1. Erstellen einer execution role

  • Öffnen Sie die roles-Seite in der IAM console.
  • Wählen Sie Create role.
  • Erstellen Sie eine role mit den folgenden properties:
  • Trusted entity – Lambda.
  • Permissions – AWSLambdaBasicExecutionRole.
  • Role name – lambda-role.

Die AWSLambdaBasicExecutionRole policy enthält die permissions, die die function benötigt, um logs in CloudWatch Logs zu schreiben. Sie können der role später permissions hinzufügen oder sie gegen eine andere role austauschen, die spezifisch für eine einzelne function ist.

2. Erstellen einer Java function

  • Öffnen Sie die Lambda console.
  • Wählen Sie Create function.
  • Configure die folgenden Einstellungen:
  • Name – my-function.
  • Runtime – Java 11.
  • Role – Choose an existing role.
  • Existing role – lambda-role.
  • Wählen Sie Create function.
  • Um ein test event zu configure, wählen Sie Test.
  • Geben Sie für den Event name test ein.
  • Wählen Sie Save changes.
  • Um die function zu invoke, wählen Sie Test.

package cc.peerapat;
import com.amazonaws.services.lambda.runtime.Context;
import com.amazonaws.services.lambda.runtime.RequestHandler;
import com.amazonaws.services.lambda.runtime.events.APIGatewayV2HTTPEvent;
import com.amazonaws.services.lambda.runtime.events.APIGatewayV2HTTPResponse;

import java.util.HashMap;
import java.util.Map;

/**
* cc.peerapat.BahtTextHTTP::handleRequest
*/
public class BahtTextHTTP
  implements RequestHandler<APIGatewayV2HTTPEvent, APIGatewayV2HTTPResponse> {

	final Map<String, String> headers = new HashMap<>();
	
	public BahtTextHTTP() {
		headers.put("Content-Type", "text/plain");
	}
	
	@Override
	public APIGatewayV2HTTPResponse handleRequest(
		final APIGatewayV2HTTPEvent request
		, final Context context) {
		return buildResponse(BahtText.toText(request.getBody()));
	}
	
	private APIGatewayV2HTTPResponse buildResponse(final String body) {
		var response = new APIGatewayV2HTTPResponse();
		response.setIsBase64Encoded(false);
		response.setStatusCode(200);
		response.setHeaders(headers);
		response.setBody(body);
		return response;
	}
}
