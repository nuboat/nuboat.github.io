# SEO and AEO Skill

## Core Principles
1.  **SEO (Search Engine Optimization)**: Optimize for traditional crawlers (Googlebot, Bingbot). Focus on keyword intent, semantic HTML, structured data, and metadata.
2.  **AEO (Answer Engine Optimization)**: Optimize for AI models and conversational engines (ChatGPT, Gemini, Perplexity, Claude). Focus on clear, concise answers, authoritative tone, Q&A formats, structured markdown, and predictable content hierarchies.

## SEO Checklist
-   **Title Tags**: Unique, descriptive, under 60 characters, with primary keyword near the beginning.
-   **Meta Descriptions**: Compelling summary, under 160 characters, including primary keyword and clear call-to-action or value proposition.
-   **Semantic HTML**: Use proper heading hierarchy (one `H1`, nested `H2`, `H3`). Use `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`.
-   **Image Optimization**: Always include descriptive `alt` text for images.
-   **Internal Linking**: Link to other relevant pages within the website using descriptive anchor text.
-   **URL Structure**: Use clean, descriptive URLs (if applicable to the platform).

## AEO Checklist
-   **Direct Answers (BLUF - Bottom Line Up Front)**: Answer the core question directly in the first paragraph or immediately after a heading.
-   **Structured Content**: Use lists (bulleted or numbered) and tables for data or step-by-step instructions. AI models prefer structured data for extraction.
-   **Natural Language & Conversational Tone**: Write content that directly addresses user queries (e.g., "How to...", "What is...").
-   **FAQ Sections**: Include an FAQ section with explicit Questions (H2/H3) and concise Answers (paragraphs).
-   **Authoritativeness & Clarity**: Avoid ambiguity. State facts clearly.

## Execution Rules
When asked to optimize or create a page with this skill:
1.  Review existing content against the SEO and AEO checklists.
2.  Suggest or implement a revised structure (e.g., moving direct answers to the top).
3.  Add or update standard metadata (title, meta description).
4.  Ensure headings clearly signpost the content.
5.  If writing a new post, ensure it follows these guidelines strictly from the first draft.

Remember: Good AEO is often excellent SEO. Clear, well-structured, high-quality content performs well on both paradigms.
