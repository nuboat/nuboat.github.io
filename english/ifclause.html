English Grammar: If-Clauses (Conditionals)

June 2026  ·  Peerapat A  ·  ← Fast English

สรุปประเด็นสำคัญ (BLUF): ประโยคเงื่อนไข (If-clause) มี 5 รูปแบบหลัก: Zero, First, Second, Third และ Mixed โดยแต่ละแบบมีการจับคู่ Tense ที่ต่างกันตามความเป็นไปได้และเวลา บทความนี้สรุปโครงสร้างพร้อมตัวอย่างและจุดที่พลาดบ่อยเพื่อให้จำง่ายและนำไปใช้ได้ทันที

ประโยคเงื่อนไข (If-clause / Conditional) คือการพูดว่า "ถ้า...แล้ว..." ทุกประโยคมีสองส่วน คือ if-clause (เงื่อนไข) และ main clause (ผลลัพธ์) หัวใจอยู่ที่การจับคู่ "กาลของกริยา" ทั้งสองฝั่งให้ถูกตามว่าเรากำลังพูดถึง เรื่องจริงเสมอ, เรื่องที่อาจเกิดในอนาคต, หรือ เรื่องสมมติที่ตรงข้ามกับความจริง

หมายเหตุการสลับที่: จะวาง if-clause ไว้หน้าหรือหลังก็ได้ ความหมายเหมือนกัน ต่างกันแค่เครื่องหมายจุลภาค — ถ้าขึ้นต้นด้วย if ให้ใส่ comma คั่นกลาง (If it rains, I stay home.) แต่ถ้า if อยู่กลางประโยคไม่ต้องใส่ (I stay home if it rains.)

1. Zero Conditional — ถ้า...แล้ว (ความจริงเสมอ)

โครงสร้าง: if + present simple, present simple

ใช้กับความจริงทางธรรมชาติ กฎทั่วไป หรือสิ่งที่เกิดขึ้นเป็นนิสัย — "ถ้าเกิด A ก็จะเกิด B เสมอ" ทั้งสองฝั่งใช้ present simple

  • If you heat ice, it melts. (ถ้าเอาน้ำแข็งไปโดนความร้อน มันจะละลาย — จริงเสมอ)
  • If I drink coffee at night, I can't sleep.
  • Plants die if they don't get water.

2. First Conditional — ถ้าสิ่งหนึ่งเกิด อีกสิ่งจะเกิดตามไป

โครงสร้าง: if + present simple, will + V1

ใช้กับเหตุการณ์ในอนาคตที่ มีโอกาสเกิดจริง — เงื่อนไขที่เป็นไปได้ จุดพลาดที่พบบ่อยคือเผลอใส่ "will" ในฝั่ง if ด้วย (ผิด!) ฝั่ง if ใช้ present simple เสมอ ส่วน will สามารถแทนด้วย can / may / should / กริยาคำสั่ง (imperative) ได้

  • If it rains tomorrow, I will stay home. (ถ้าพรุ่งนี้ฝนตก ฉันจะอยู่บ้าน — เป็นไปได้)
  • If you study hard, you will pass the exam.
  • If you see him, tell him to call me. (ฝั่งผลใช้ประโยคคำสั่งได้)
  • ✗ If it will rain, I will stay home. (ห้ามใส่ will ในฝั่ง if)

3. Second Conditional — เรื่องสมมติตรงข้ามกับความจริงปัจจุบัน

โครงสร้าง: if + past simple, would + V1

ใช้สมมติเรื่องที่ ตรงข้ามกับความจริงในปัจจุบัน หรือแทบเป็นไปไม่ได้ — "ถ้าเป็นอย่างนั้น (ซึ่งมันไม่ใช่) ก็คงจะ..." แม้พูดถึงปัจจุบันแต่ใช้ past simple ในฝั่ง if และนิยมใช้ "were" กับทุกประธาน (If I were you...) นอกจากนี้ would ยังใช้ ขอร้องอย่างสุภาพ ได้ด้วย

  • If I had a million baht, I would travel the world. (ความจริงคือไม่มี — แค่สมมติ)
  • If I were you, I would apologize. (ใช้ were กับทุกประธาน)
  • If she studied more, she would get better grades. (ความจริงคือเธอไม่ค่อยอ่าน)
  • It would be great if you could help me. (ขอร้องเชิงสุภาพ)

4. Third Conditional — เรื่องสมมติตรงข้ามกับความจริงในอดีต

โครงสร้าง: if + past perfect, would have + V3

ใช้สมมติเรื่องในอดีตที่ เกิดไปแล้วและแก้ไขไม่ได้ — "ถ้าตอนนั้นเป็นอย่างนั้น (ซึ่งมันไม่ได้เป็น) ผลก็คงจะต่างไป" มักใช้แสดงความเสียดายหรือตำหนิย้อนหลัง ฝั่ง if ใช้ "had + V3" ฝั่งผลใช้ "would have + V3"

  • If I had studied harder, I would have passed the exam. (ความจริงคือไม่ได้ตั้งใจ เลยสอบตก)
  • If you had told me, I would have helped you. (แต่เธอไม่บอก ฉันเลยไม่ได้ช่วย)
  • She would have come if we had invited her.

5. Mixed Conditional — เงื่อนไขอดีต ผลในปัจจุบัน

โครงสร้าง: if + past perfect, would + V1 (+ now)

ใช้เมื่อ ข้างหน้า (if) ตรงข้ามกับอดีต แต่ ข้างหลัง (ผล) ตรงข้ามกับปัจจุบัน/อนาคต — คือเหตุเกิดในอดีต แต่ผลส่งมาถึงตอนนี้ มักมีคำว่า "now" หรือ "today" เพื่อย้ำว่าเป็นปัจจุบัน

  • If I had saved money, I would be rich now. (อดีตไม่ได้เก็บเงิน → ตอนนี้เลยไม่รวย)
  • If she had taken the job, she would live in Tokyo today.
  • If they had caught the train, they would be here by now.

สรุปตารางเทียบ

ชนิดif-clausemain clauseใช้เมื่อ
Zeropresent simplepresent simpleความจริงเสมอ / กฎทั่วไป
Firstpresent simplewill + V1อนาคตที่เป็นไปได้จริง
Secondpast simplewould + V1สมมติตรงข้ามปัจจุบัน / ขอร้องสุภาพ
Thirdpast perfectwould have + V3สมมติตรงข้ามอดีต
Mixedpast perfectwould + V1 (now)เหตุในอดีต ผลในปัจจุบัน

จุดที่คนพลาดบ่อย

  • อย่าใส่ will/would ในฝั่ง if — ฝั่ง if ใช้ tense ปกติเสมอ (present/past/past perfect)
  • First vs Second: First = เป็นไปได้จริง (present + will), Second = สมมติ/ตรงข้ามความจริง (past + would)
  • Second vs Third: Second พูดถึงปัจจุบัน (past simple), Third พูดถึงอดีต (past perfect + would have)
  • If I were ไม่ใช่ "If I was" ในภาษาทางการ
  • Mixed: ดูเวลาของเหตุกับผลให้ดี — เหตุอดีต (had + V3) แต่ผลปัจจุบัน (would + V1)

วิธีจำง่ายที่สุดคือถามตัวเองก่อนว่า "เรื่องนี้จริงได้ไหม และอยู่ในเวลาไหน" แล้วค่อยเลือกสูตร — ฝึกแต่งวันละ 2–3 ประโยคต่อชนิด อีกไม่นานจะเลือกใช้ได้อัตโนมัติ
