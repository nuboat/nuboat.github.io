English Grammar: Purpose, Result & Degree

June 2026  ·  Peerapat A  ·  ← Fast English

โครงสร้างไวยากรณ์สามกลุ่มที่เจอบ่อยที่สุดในข้อสอบและการเขียนจริง คือการบอก จุดประสงค์ (Purpose) ว่า "ทำสิ่งนี้เพื่ออะไร", การบอก ผลลัพธ์ (Result) ว่า "มากเสียจนกระทั่งเกิดอะไรขึ้น" และการบอก ระดับ (Degree) ว่า "มากเกินไป" หรือ "เพียงพอ" หน้านี้รวบรวมรูปแบบทั้งหมด พร้อมสูตรและตัวอย่างให้เห็นภาพชัด ๆ ในที่เดียว

1. Clauses of Purpose — เพื่อที่จะ

ใช้บอกว่าทำกริยาหนึ่ง เพื่อจุดประสงค์ อะไร ภาษาไทยคือ "เพื่อที่จะ" มีสองกรณีหลัก คือ ประโยคเดียว (ประธานเดียวกัน ใช้ to-infinitive) กับ สองประโยค (ประธานต่างกัน ต้องมี subject + กริยาช่วย)

1.1 รูปประโยคเดียว: to / in order to / so as to + V1

ทั้งสามแบบแปลว่า "เพื่อที่จะ" และแทนกันได้ "to" เป็นกลางและไม่เป็นทางการที่สุด ส่วน "in order to" และ "so as to" เป็นทางการกว่า ตามด้วยกริยาช่องที่ 1 (V1) เสมอ

  • She studies hard to pass the exam. (เธอตั้งใจเรียนเพื่อสอบผ่าน)
  • He woke up early in order to catch the first train.
  • We left quietly so as to avoid waking the baby.

1.2 รูปปฏิเสธ: in order not to / so as not to + V1

แปลว่า "เพื่อที่จะไม่" สังเกตว่า "not" วางไว้หน้า "to" (ห้ามใช้ "to not") และ ไม่ นิยมใช้ "not to" เดี่ยว ๆ

  • She whispered in order not to disturb the others. (เธอกระซิบเพื่อจะไม่รบกวนคนอื่น)
  • I wrote it down so as not to forget the address.

1.3 รูปสองประโยค: in order that / so that + S + กริยาช่วย + V1

ใช้เมื่อประธานของสองประโยค ต่างกัน หรือเมื่อต้องการกริยาช่วย (can/could, will/would, may/might) หลัง "so that" / "in order that" ต้องตามด้วย ประโยคเต็ม (subject + verb) เสมอ

  • I'll explain it again so that everyone can understand. (ฉันจะอธิบายอีกครั้งเพื่อให้ทุกคนเข้าใจ)
  • She spoke slowly in order that the students could take notes.
  • Turn off your phone so that it won't disturb the class. (รูปปฏิเสธ ใช้ won't / wouldn't)
เลือกใช้เมื่อโครงสร้าง
ประธานเดียวกันto / in order to / so as to + V1
ประธานต่างกัน หรือมีกริยาช่วยso that / in order that + S + can/could/will/would + V1

2. Clauses of Result & Degree — มากเสียจนกระทั่ง / มากเกินไป / เพียงพอ

กลุ่มนี้บอกว่า "มาก/น้อย" ในระดับหนึ่ง จนทำให้ เกิดผลตามมา หัวใจอยู่ที่การเลือกคำให้ถูกตามชนิดของคำที่ตามมา (adjective/adverb หรือ noun, นับได้หรือนับไม่ได้)

2.1 so + adj/adv + that + S + V — มากเสียจนกระทั่ง

ใช้ "so" กับคำคุณศัพท์ (adjective) หรือคำกริยาวิเศษณ์ (adverb) แล้วตามด้วย "that" + ประโยคผลลัพธ์

  • The box was so heavy that I couldn't lift it. (กล่องหนักมากจนยกไม่ขึ้น)
  • He ran so fast that no one could catch him.

2.2 such (a) + adj + N + that + S + V — มากเสียจนกระทั่ง

ความหมายเหมือน "so...that" แต่ใช้กับ คำนาม ข้อแตกต่างสำคัญ: "so" + adj เดี่ยว ๆ / "such" + (a/an) + adj + noun

  • It was such a heavy box that I couldn't lift it. (นามนับได้เอกพจน์ ใส่ a)
  • They were such kind people that everyone loved them. (นามพหูพจน์ ไม่ใส่ a)
  • เทียบ: It was so heavy that... = It was such a heavy box that...

2.3 too + adj/adv + to + V1 — มากเกินไปที่จะ

"too" มีความหมายเชิงลบเสมอ คือ "มากเกินไปจนทำไม่ได้" ตามด้วย to-infinitive (อย่าสับสนกับ "very" ที่แค่เน้นว่า "มาก" เฉย ๆ)

  • This coffee is too hot to drink. (กาแฟร้อนเกินไปที่จะดื่ม)
  • He spoke too quietly to be heard.
  • เพิ่มผู้กระทำได้ด้วย "for someone": The box is too heavy for me to lift.

2.4 so + many/few + พหูพจน์นับได้ + that — มากเสียจนกระทั่ง

เมื่อตามด้วย คำนามนับได้พหูพจน์ (countable plural) ใช้ "so many" (มาก) หรือ "so few" (น้อย)

  • There were so many people that we couldn't find a seat. (คนมากจนหาที่นั่งไม่ได้)
  • She made so few mistakes that she got the highest score.

2.5 so + much/little + นามนับไม่ได้ + that — มากเสียจนกระทั่ง

เมื่อตามด้วย คำนามนับไม่ได้ (uncountable) ใช้ "so much" (มาก) หรือ "so little" (น้อย)

  • He had so much work that he stayed up all night. (งานมากจนต้องอดนอนทั้งคืน)
  • There was so little water that the plants died.

2.6 adj/adv + enough (for someone) + to + V1 — เพียงพอที่จะ

"enough" วาง หลัง คำคุณศัพท์/กริยาวิเศษณ์ แปลว่า "พอที่จะ"

  • She is old enough to drive. (เธอโตพอที่จะขับรถได้)
  • He didn't run fast enough to win.
  • The rope is long enough for us to reach the window.

2.7 enough + N + (for someone) + to + V1 — เพียงพอที่จะ

แต่เมื่อใช้กับ คำนาม ต้องวาง "enough" ไว้ หน้า คำนาม

  • We have enough money to buy the tickets. (เรามีเงินพอที่จะซื้อตั๋ว)
  • There aren't enough chairs for everyone to sit.

สรุปจุดที่คนพลาดบ่อย

  • so vs such: "so" + adj/adv, "such" + (a) + adj + noun
  • many/few vs much/little: นับได้พหูพจน์ใช้ many/few, นับไม่ได้ใช้ much/little
  • too vs enough: too = "เกินไป" (ทำไม่ได้), enough = "พอ" (ทำได้)
  • ตำแหน่ง enough: หลัง adj/adv แต่หน้า noun
  • not วางตรงไหน: in order not to / so as not to (ไม่ใช่ to not)

จำสูตรให้แม่น แล้วฝึกแต่งประโยคของตัวเองวันละ 2–3 ประโยคต่อโครงสร้าง อีกไม่นานทั้งสามกลุ่มนี้จะกลายเป็นเรื่องอัตโนมัติ — เห็นโจทย์ปุ๊บ รู้คำตอบปั๊บ
