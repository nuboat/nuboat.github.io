## ifclause
- if + present simple, present simple = ถ้า แล้ว
- if + present simple + will + V1 = ถ้าสิ่งหนึ่งเกิด อีกสิ่งจะเกิดตามไปด้วย
- if + present simple + would + V1 = ใช้ตรงข้ามกับความจริงในปัจจุบัน / ใช้ขอร้องเชิงสุภาพ
- if + past perfect + would have + V3 = ใช้ตรงข้ามกับความจริงในอดีต
- if + past perfect + would + V1 + now = ข้างหน้าตรงข้ามกับอดีต ข้างหลังตรงข้ามกับปัจจุบัน/อนาคต

## Purpose 
- to / in order to / so as to + V1 = เพื่อที่จะ
- in order not to / so as not to + V1 = เพื่อที่จะไม่
- in order that /so that + S + Vช่วย + V1 = เพื่อที่จะ (กรณี 2 ประโยค)
- soadj/adv + that + S + V = มากเสียจนกระทั่ง
- too + adj/adv + to + V1 = มากเกินไปที่จะ
  such (a) + adj + N + that + S + V = มากเกินไปที่จะ
- so + many/few + plu CN + that + S + V = มากเสียจนกระทั่ง
  so + much/little + sin UN + that + S + V = มากเสียจนกระทั่ง
- adj/adv + enough (for someone) + to + V1 = เพียงพอที่จะ
  enough + N + (for someone) + to + V1 = เพียงพอที่จะ
